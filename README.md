### Under the Hood
* **State & Logic:** Custom JavaScript handles the dynamic modal layout, animated skill components, and interactive triggers based on user engagement.
* **Styling Engine:** Raw CSS powered by a strict `:root` variable system. I built custom keyframes for the gradient color-sweeps and neon-hover glow effects natively.
* **Layout:** A heavy mix of CSS Grid and Flexbox ensures everything snaps instantly across mobile and desktop.
* **Effects:** Intensive use of `backdrop-filter` for glassmorphic cards and floating orbs to establish real z-index depth.

## 🚀 Running It Locally

This is a static site. No `npm install`, no endless build steps, no waiting.

1. Clone it down:
   ```bash
   git clone https://github.com/MRINALPRAKASHFSD/MY_PORTFOLIO.git
   ```
2. Open `INDEX.HTML` directly in any browser. That's it.
